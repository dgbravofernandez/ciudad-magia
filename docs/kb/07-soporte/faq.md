# FAQ — base para ayuda in-app

**Owner:** Diego · **Última actualización:** 2026-06-20

Cada duda recurrente de un cliente va aquí. Cuando una entrada se repite, exponerla como ayuda in-app (tooltip, drawer o página de ayuda). Fuente para deflexión de soporte ([07](CONTEXT.md)).

> Estado inicial: a poblar con dudas reales de Getafe. No inventar preguntas que nadie ha hecho.

## Estructura sugerida por temas

### Primeros pasos
- _(p. ej.: "¿Cómo doy de alta mi primer equipo?")_

### Pagos y cuotas
- _(p. ej.: "¿Cómo registro un pago parcial?", "¿Cómo hago el cierre de caja?")_

### Jugadores e inscripciones
- _(p. ej.: "¿Cómo comparto el formulario de inscripción con las familias?")_

### Comunicaciones
- _(p. ej.: "¿Por qué no llega un email?" → revisar config remitente)_

### Integración federación (RFFM)
- _(p. ej.: "¿Cada cuánto se actualizan las actas/goleadores?")_

### Cuenta y facturación
- _(p. ej.: "¿Cómo cambio de plan / cancelo?")_
